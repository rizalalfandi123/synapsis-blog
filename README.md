## Development

- **Make sure you install node js version 18 or latest**
- **Installing App**
  ```bash
  pnpm install
  # or
  yarn install
  # or
  npm install
  ```
- **Run development Server (http://localhost:3000)**
  ```bash
  pnpm dev
  # or
  yarn dev
  # or
  npm run dev
  ```

## Build

**To build app, please run**

```bash
 pnpm build
 # or
 yarn build
 # or
 npm run build
```

see more in [Next JS Documentation](https://nextjs.org/docs)
